👗 Voluble Boutique
A modern e-commerce platform for a fashion-forward clothing brand.

📐 Original Design: Figma Prototype


🧠 About the Project
Voluble Boutique is a full-featured e-commerce storefront built for a fashion brand. The platform focuses on delivering a clean, modern shopping experience — from product discovery through checkout — with a strong emphasis on visual design, brand identity, and conversion-optimized flows.

This project was built as a complete product concept with a live frontend and Supabase backend.


✨ Features
🛍️ Product Catalog — Browse and filter clothing collections by category, size, and style
🔍 Product Discovery — Search and explore the full product range with rich imagery
🛒 Shopping Cart — Add, remove, and manage items before checkout
💳 Checkout Flow — Streamlined purchase flow optimized for conversion
👤 User Authentication — Secure account creation and login via Supabase Auth
📦 Order Management — Track and manage customer orders
📱 Responsive Design — Fully optimized for mobile and desktop shopping

🚀 Running Locally
# Clone the repository

git clone https://github.com/davidolivo5999/Ecommercewebsiteforproducts.git

# Navigate into the project

cd Ecommercewebsiteforproducts

# Install dependencies

npm install

# Start the development server

npm run dev

Then open http://localhost:5173 in your browser.
